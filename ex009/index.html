Listas Ordenadas e Não Ordenadas

Listas Ordenadas

Listas Ordenadas são quando a ordem da lista importa, onde/quando a ordem das coisas que seram listadas é unma coisa muito essencial

Para criar uma lista ordenada se usa a tag ol e para identificar cada item da lista se ultiliza a tag li

Exemplos

  1. Primeiro Item
  2. Segundo Item
  3. Terceiro Item
  4. Quarto Item

É importante dizer que o fechamento da tag ol é obigatoria, mas o fechamentp= da tag li é opicional

identificadores

A tag ol tem um parametro type, ou seja, pode se incluir o tipo de identificador que vc quer nas suaas listas

ex. < ol type = I >

  1. 1 - O padrão. faz listas numeradas ex. 1,2,3 etc
  2. A - Listas alfabeticas em maiusculo
  3. a - LIstas alfabeticas em minusculo
  4. I - Listas em algarismo romano maiusculo
  5. i - Listas em algarismo romano minusculo

Listas não Ordenadas

Quando a ordem nao importa

usa a tag ul para definir,e a li para listar

  • Item 1
  • Item 2
  • Item 3

Do mesmo jeito que as ordenadas, as nao ordenadas também tem tipos, que são eles disc, circle, square

Também se pode misturar as coisas

Para se misturar, basta fazer listas dentro de listas

  1. item 1
  2. item 2
    1. item 2.1
    2. item 2.2
  3. item 3
  4. item 4
